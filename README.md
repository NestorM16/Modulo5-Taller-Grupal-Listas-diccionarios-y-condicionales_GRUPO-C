# Programa de Registro de Personas

## Descripción

Este proyecto consiste en un programa desarrollado en Python que permite consultar y registrar personas utilizando listas, diccionarios y estructuras de control.

El programa permite:
- Buscar una persona y mostrar su edad.
- Mostrar la lista de personas registradas.
- Agregar nuevas personas al registro.
- Salir del programa mediante un menú de opciones.

## Tecnologías utilizadas

- Python 3

## Conceptos utilizados

En este proyecto se aplican los siguientes conceptos:

- Listas para almacenar nombres de personas.
- Diccionarios para relacionar nombres con edades.
- Condicionales (`if`, `elif`, `else`) para tomar decisiones.
- Ciclo `while` para mantener el menú activo.
- Ciclo `for` para recorrer la lista de personas.
- Entrada de datos mediante `input()`.
